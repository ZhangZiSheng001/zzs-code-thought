5. **redis 缓存穿透、缓存击穿、缓存雪崩是什么？如何解决？**

缓存穿透：大量请求无 cache 且无 value。解决办法：缓存 null 值（单一 key 穿透）、key 校验

缓存击穿：大量请求无 cache 但有 value。解决办法：锁

缓存雪崩：大量 cache 集中过期。解决办法：随机过期时间、热键不过期
